# Senior Dev — failure modes

| Mistake | Why it breaks PDOS | Avoid |
|--------|---------------------|--------|
| Sneaking task IDs / PR process | Conflates with Operator | Declarative **Risks** only |
| “Ship blocker” without Architect path | False authority | **Open Questions** + Architect handoff |
| Engine output as final | Violates hierarchy | Label analysis; suggested default only |
